# Teruvion Product Roadmap & Valuation Framework

## Vision

**Mission**: Build a living Digital Earth for understanding, exploring, and governing the planet.

**Positioning**: AI-native Digital Earth Intelligence Platform

**Analogy**: Palantir × Esri/ArcGIS × AlphaSense × Digital Earth

---

## What Makes It NOT a Small Tool

Teruvion is NOT a small tool if it achieves:

1. **It builds Digital Earth object graph**, not just summarizes sources
2. **Multi-source integration**: paper/repo/dataset/report/news → unified object layer
3. **Provenance**: Every object and relation is traceable to source
4. **Long-term graph memory**: Not one-shot outputs
5. **Capability ontology**: Data, observation, modeling, computing, governance, action
6. **World ontology**: Resources, risks, processes, events, human activities, infrastructure
7. **Lens recomposition**: Map/evidence/process/risk/resource/scenario/action views
8. **Continuous exploration**: Monitoring and discovery, not one-shot Q&A

### Bear Case: Small Tool
- URL input → AI summary → paper/dataset cards → simple graph
- Result: Small tool, $1M–10M valuation, easily eaten by LLMs

### Base Case: Vertical Platform

- Multi-source object graph, evidence/provenance, monitoring

- Unified resource/risk/model/action capabilities

- Result: Infrastructure company, $1B–10B+ valuation
